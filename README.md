# JG Planos de Saúde

Landing page focada em planos de saúde empresariais para famílias com CNPJ ou MEI.

O arquivo `responsive.css` contém os ajustes específicos para celulares e
tablets e deve permanecer na mesma pasta do `index.html`.

## Publicação
Envie todos os arquivos deste projeto para a raiz do GitHub. O Netlify deve publicar a raiz do repositório.
